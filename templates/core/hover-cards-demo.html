Available Hover Classes

  • hover-reveal-card - Slide reveal from bottom
  • hover-slide-up - Enhanced slide up
  • hover-slide-left - Content slides from left
  • hover-tilt-card - 3D tilt effect
  • hover-overlay-card - Full overlay reveal
  • hover-glow-card - Animated gradient border
  • hover-shine-card - Light sweep effect
  • hover-morph-card - Shape morphing
  • hover-depth-card - 3D depth rise
  • hover-preview-card - Content preview
  • hover-icon-reveal - Icon animation
  • hover-stagger-reveal - Staggered items

Tip: Combine multiple classes for unique effects! Try hover-reveal-card hover-slide-up hover-zoom-container together.
